# GreenLedger-Eco-Friendly-Accounting-Website
GreenLedger is a responsive, eco-friendly accounting website showcasing services, sustainability practices, and client-focused solutions. Built with HTML, CSS, and Bootstrap, it aims to demonstrate clean UI design, modern layout skills, and digital-first accounting concepts.

# Features

Responsive layout using Bootstrap
Clean navbar and hero section
Services section with icons
About section promoting sustainability
Contact form for user queries
Professional green-themed UI

# CSS3
